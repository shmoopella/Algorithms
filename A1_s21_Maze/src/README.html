Maze


Программа генерирует и отрисовывает лабиринты.

В программе предусмотрена кнопка для загрузки лабиринта из файла. Загруженный лабиринт отрисовывается на экране в поле размером 500 x 500 пикселей. Максимальный размер лабиринта - 50х50.

Предусмотрена возможноть автоматической генерации идеального лабиринта по алгоритму Эллера (из любой точки можно попасть в любую другую точку ровно одним способом, не имеет изолированных областей и петель). Для генерации идеального лабиринта понадобится ввести его размерность: количество строк и столбцов.

Предоставлена функция демонстрации решения лабиринта, изображенного на экране. Если указаны некорректные параметры или загружен неидеальный лабиринт и между указанными точками нельзя найти путь, решение не будет отрисовано.
